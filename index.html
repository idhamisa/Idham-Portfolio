PLC Rejection Integrated System Based on Deep Learning

Developed an automated rejection system for can cap defects using AI and PLC integration. Applied YOLOv8 for real-time defect detection and classification, deployed on Raspberry Pi 5 with Hailo AI Hat, and controlled PLC actuators for defective product rejection.

Smart Pest Repellent Using Ultrasonic Wave

Designed an ultrasonic-based pest repellent system using ESP32-CAM and Arduino Mega. Integrated IoT monitoring to provide live camera feed and real-time control over the device through a web interface.

Roadtax Expiry Detector

Implemented a smart system that automatically detects roadtax expiry using OCR and AI. Captures vehicle number plates, extracts relevant information, and alerts users if their roadtax is nearing expiry. Designed to enhance automation for vehicle management and compliance monitoring.
